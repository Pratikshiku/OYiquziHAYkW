### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户（党员）、党支部、管理员。

**普通用户功能：**
1. 登录
2. 首页
3. 党建要闻
4. 党建地图
5. 党建学习
6. 党建活动
7. 问卷调查
8. 民主投票
9. 学习测试
10. 后台管理
11. 个人中心
12. 考试记录
13. 错题本
14. 我的收藏


**党支部功能：**
1. 登录
2. 修改密码
3. 个人信息
4. 党员管理
5. 党费缴纳管理
6. 党建活动管理
7. 活动报名管理



**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 党员管理(年龄统计、学历统计)
4. 党支部管理
5. 党建要闻管理
6. 党建地图管理
7. 党建学习管理
8. 学习心得管理
9. 党费缴纳管理
10. 党建活动管理
11. 活动报名管理
12. 问卷调查管理
13. 问卷回答管理
14. 民主投票管理
15. 投票信息管理
16. 试题管理
17. 学习测试管理
18. 系统管理
19. 考试管理（学习测试列表、考试记录、错题本）
